Software Engineering — Lab instructions

Spring 2014

Introduction

This document explains the general structure of the laborations and how you should approach them. As you might already know, this course consist of four blocks. Covering different topics. The laborations consequently fit within the same blocks.

Each block has one laboration document. Each laboration document, however, contain multiple exercises.

Each block has multiple laboration sessions in the schedule. Naturally this is because each laboration document contains more laborations than what is exected of you to finish in one session. This means you need to plan your time. When you first encounter a laboration document. Read through the whole thing. Make mental notes on which exercises that are mandatory and which are not. And plan your time accordingly.

Deliverables

Each laboration document specifies its deliverables. To complete a lab you simply have to...

  1. Complete a deliverable
  2. Show your work to the laboration assistant and follow up on any comments he/she might give untill you get an OK.
  3. Repeat 1-2 until enough (defined below) deliverables have been assesed to pass the block.
  4. Follow the handin instructions (defined below) and upload the whole block as a zip-file to Studentportalen

For each block you must finish all mandatory exercises , and at least 1 optional.

How to hand in...

  • Each block have it's own assignment area on Studentportalen.
  • Each block must be handed in (uploaded to Studentportalen) as a zip-file (.zip or .rar only!)
  • The handed in zip-file of a block must contain a (clearly named) folder for each deliverable you wish to deliver.

Code conventions...

  • All code must be properly indented.
  • Sensible naming conventions must be practiced.

Grading

Labs will only be graded as pass or fail.

Each block needs to be handed in a short time after the end of the block. The deadline for each block handin can be found at the corresponding assignment area on Studentportalen. Make sure you are aware of when the deadline is when planning your work.

If you fail a lab you can of course re-submit and supplement. The last date for re-submitting can be found on Studentportalen under the corresponding assignment area. Generally it should be at the end of the course. But make sure you are aware of the last date for supplementation if the situation would occur.

If you fail a lab, and fail (or ignore) to re-submit, we will not mark you laboration until the end of the semester. Which means you will not be able to recieve your grade in a timely manner.
