Software Engineering — Lab instructions

Spring 2014

Introduction

This document explains the general structure of the laborations and how you should approach them. As you might already know, this course consist of four blocks. Covering different topics. The laborations consequently fit within the same blocks.

Each block has one laboration document. Each laboration document, however, contain multiple more or less related exercises. Each block has multiple laboration sessions in the schedule. Naturally this is because each laboration document contains more laborations than what is exected of you to finish in one session.

This means you need to plan your time. When you first encounter a laboration document. Read through the whole thing. Make mental notes on which exercises that are mandatory and which are not. And plan your time accordingly.

Deliverables

Each laboration document specifies its deliverables. To complete a lab you simply have to...

  1. Complete a deliverable
  2. Show your work to the laboration assistant and follow up on any comments he/she might give untill you get an OK.
  3. When enough deliverables of a block has been assessed — follow the handin instructions below and upload the whole block as a zip-file to Studentportalen

How to hand in...

  • Each block have it's own assignment area on Studentportalen.
  • Each block must be handed in (uploaded to Studentportalen) as a zip-file (.zip or .rar only!)
  • The handed in zip-file of a block must contain a (clearly named) folder for each deliverable you wish to deliver.

Code conventions...

  • All code must be properly indented.
  • Sensible naming conventions must be practiced.

Grading

The laborations will play a 10% role in your overall grade for this course. You can pass or not pass the laborations. Both each laboration, as well as the laborations as a whole.

Each passed laboration translates into a number of points. You need to achieve a total number of points in order to be able to get a pass for the laborations. However, this means that you can choose to skip some laborations and focus on the ones you really want to complete.

  • In total there’s 100 lab points.
  • Each lab block has 25 points.
  • To get a pass for all labs, you need a total of 75 points.
